

# CSWALA Frequently Asked Question Page

A responsive Frequently Asked Questions (FAQ) page for CSWALA, an online computer science education platform. The page features an interactive accordion-style FAQ section, a modern header, and a responsive footer.

## Features

- Interactive accordion-style FAQ sections
- Responsive design that works on desktop and mobile devices
- Modern and clean user interface
- Animated hover effects
- Navigation header with menu items
- Comprehensive footer with contact information and social links
- Custom styling with smooth transitions and animations

## Technologies Used

- HTML
- CSS
- JavaScript
- Font Awesome Icons
- Google Fonts (Roboto)
